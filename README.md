# SikshyaX — Gamified Education Platform (Frontend Demo)

Frontend prototype for **SikshyaX**, a gamified education platform for Nepal focused on competitive exams and multiplayer MCQ battles.

## Experiences

| Portal | Demo login | Features |
|--------|------------|----------|
| **Student App** | `student@demo.np` / `demo123` | Home, Play, Practice, AI Tutor, Profile (bottom nav) |
| **Admin Panel** | `admin@sikshyax.com` / `demo123` | Questions, users, tournaments, revenue |

Phone OTP demo: enter any number → OTP `123456`

## Student App Screens

- **Home** — Rating, rank, streak, exam categories (CEE, EPS Korea, Loksewa, Engineering), tools
- **Play** — Quick Game, Friend Match, Online Match, Bot Match, Tournaments (Premium)
- **Practice** — Unlimited free MCQ practice by subject/chapter
- **AI Tutor** — ChatGPT-style exam assistant
- **Profile** — Stats, notes, routine, daily hub, subscription
- **Game** — Live MCQ battle with timer, scores, result & review
- **Leaderboard** — Monthly rankings with prizes
- **Daily Challenge** — Streak rewards
- **Daily Hub** — Nepali calendar, weather, news, rashifal, NEPSE
- **Subscription** — Plans with eSewa, Khalti, IME Pay

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:5173
